macam macam selector

  1. id selector

    2. selector yang menggunakan attribute ID dan tanda (#) diawal pada cssnya

  2. class selector

    3. selector yang menggunakan attribute class dan tanda (.)pada awal cssnya

  3. element/type selector

    4. selector yang menggunakan nama element htmlnya

  4. universal selector

    5. selector yang menggunakan tanda bintang (*) pada cssnya

  5. grouping selector

    6. selector yang menggabungkan beberapa selector menjadi satu

    contoh : h1, h2, p { color: blue; }

    akan membuat semua elemen h1, h2, dan p berwarna biru

    contoh : .class1, #id1 { font-size: 16px; }

    akan membuat elemen dengan class "class1" dan id "id1" memiliki ukuran font 16px

    contoh : div, span { margin: 10px; }

    akan memberikan margin 10px pada semua elemen div dan span

  6. attribute selector

    7. selector yang memilih elemen berdasarkan atributnya

    contoh : a[href] { color: red; }

    akan membuat semua elemen dengan atribut href berwarna merah

    contoh : input[type="text"] { border: 1px solid black; }

    akan memberikan border pada semua elemen input dengan tipe text

  7. pseudo class selector

    8. selector yang memilih elemen berdasarkan keadaan atau posisinya

    contoh : a:hover { color: green; }

  8. pseudo element selector

    9. selector yang memilih bagian tertentu dari elemen

    contoh : p::first-letter { font-size: 200%; }

    akan membuat huruf pertama dari setiap paragraf berukuran 200%

    contoh : div::before { content: "Note: "; color: blue; }

    akan menambahkan teks "Note: " sebelum setiap elemen div dengan warna biru

    contoh : li::after { content: " - item"; }

    akan menambahkan teks " - item" setelah setiap elemen li

    contoh : h1::first-line { text-transform: uppercase; }

    akan mengubah semua huruf pada baris pertama dari setiap elemen h1 menjadi huruf kapital

    contoh : blockquote::marker { color: gray; }

    akan mengubah warna marker (tanda kutip) pada elemen blockquote menjadi abu-abu

  9. combinator selector

    10. selector yang menggabungkan beberapa elemen berdasarkan hubungannya

    contoh : div > p { color: purple; }

    akan membuat semua elemen p yang merupakan anak langsung dari elemen div berwarna ungu

    contoh : ul li { list-style-type: square; }

    akan mengubah jenis bullet pada semua elemen li yang berada di dalam elemen ul menjadi kotak

  10. negation selector

    11. selector yang memilih elemen yang tidak sesuai dengan kondisi tertentu

    contoh : input:not([type="submit"]) { background-color: lightgray; }

    akan memberikan latar belakang abu-abu muda pada semua elemen input kecuali yang bertipe submit
